## 维度树

```
·  0D   r=3           |Aut|=168     组合骨架
│       7点·7线        frame=28     无距离
│
—  1D   d₀=2π/3       S¹/3等分     距离量子
│       φ=120°                     离散/连续分叉
│
╱╲ 2D   -1/3  ⊥  1/√3              角度分裂
│       π/3       π/4              不可通约
│       6+6边     3+6边            Hub三层耦合
│
◆  3D   d=3            α+2β=π      冻结
        gap=(√3−1)/3   Δ/Λ=2π     ∂²=0保护
```

| 维度 | 根常数 | √d | 几何性质 |
|---|---|---|---|
| 0D | r=3 | 0 | 无距离（组合） |
| 1D | d₀=2π/3 | 1 | 距离 |
| 2D | -1/3, 1/√3 | √2 | 角度（sin(arccos(-1/3))=2√2/3） |
| 3D | α+2β=π | √3 | 手征（1/√3坐标；共轭比2−√3） |

纵向：∂逐维增加结构（组合→距离→角度→冻结）。

2D处r=3分裂为两个常数：-1/3（离散面，n=r+1=4）和1/√3（连续面，√d）。T_odd/T_even是被这两个常数选择的顶点集，非根节点。

两条路径在-1/3自指点汇聚：

$$n=4 \xrightarrow{-1/(n-1)} -1/3 \xleftarrow{\cos(d_0\sqrt{K})} r=3$$

∂从2D到3D = 给单形加一个顶点：$-1/(r{-}1) \xrightarrow{\text{球面余弦}} -1/r = -1/((r{+}1){-}1)$。

## §3D · d = 3 冻结

### 维度锁定

$T_\text{odd}$ 角 $\alpha := \arccos(1/d)$，$T_\text{even}$ 角 $\beta := \arccos(1/\sqrt{d})$。

$$\boxed{\arccos\frac{1}{d} + 2\arccos\frac{1}{\sqrt{d}} = \pi}$$

验证（d=3）：$\cos\alpha = 1/3,\; \cos\beta = 1/\sqrt{3},\; \cos 2\beta = -1/3 = -\cos\alpha$。

$$\cos(\alpha+2\beta) = -1/9 - 8/9 = -1 \quad\checkmark$$

d=3唯一正整数解：特征方程 $d^2(d{-}3)^2 = 0$。

### 减半恒等式

$$\arccos(-1/3) = 2\arccos(1/\sqrt{3})$$

证明：$\cos(\theta/2) = \sqrt{(1+\cos\theta)/2} = \sqrt{(1-1/3)/2} = 1/\sqrt{3}$。$\quad\square$

### 共轭折叠

$$g_\pm := \frac{\sqrt{d} \pm 1}{d}, \quad g_- \cdot g_+ = \frac{d-1}{d^2}$$

级联 $\cdots \to \sqrt{3} \to 3 \to 9 \to \cdots$，d=3是整数地板。$(g_- + g_+)^2 = 4/d$ 当且仅当 d=3。

### 高度比倒数

e₇到T_even（北）与T_even到(000)（南）：

$$\frac{h_\text{north}}{h_\text{south}} = \frac{1+1/\sqrt{3}}{1-1/\sqrt{3}} = 2+\sqrt{3} = \left(\frac{g_-}{g_+}\right)^{-1}$$

(000)缺席 → 体积坍缩比 = 共轭破缺比的倒数。

### 双透镜间距

$$\text{gap}(d) := \frac{1}{\sqrt{d}} - \frac{1}{d}$$

| d | gap(d) | α+2β | 闭合 |
|---|---|---|---|
| 2 | 0.207 | 5π/6 | ✗ |
| 3 | 0.244 | π | ✓ |
| 4 | 0.250 | >π | ✗ |

gap'(d)=0 ⇒ d=4（极值0.250）。角度锁选择d=3（0.244），锁在非极值。

### ∂²=0 保护

$H^2$ 穿越代价 = z-gap = $(\sqrt{3}{-}1)/3$。$H^3$ 链接代价 = $3/(\sqrt{3}{-}1)$。

$$H^2 \times H^3 = 1 \quad\Rightarrow\quad \Delta/\Lambda = 2\pi$$

$(\sqrt{3}{-}1)/3$ 影响 $\sigma$ 绝对标度，不影响 $\Delta/\Lambda$ 比值。

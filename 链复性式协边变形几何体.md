# ═══ I. ∂∂=0 ═══

## §1 区分的幂零性

S⁰ = ∂(∞|NULL)。区分 = 边界操作，边界的边界为零：

$$\partial \circ \partial = 0$$

非公设——S⁰ = ∂D¹, ∂S⁰ = ∂∂D¹ = 0。∂∂=0是区分的结构性质。S⁰同时携带拓扑（离散二值{±1}）与几何（嵌入度规）——互锁始于0D。

由此产生链复形

$$C_3 \xrightarrow{\partial_3} C_2 \xrightarrow{\partial_2} C_1 \xrightarrow{\partial_1} C_0$$

幂零蕴含 im(∂ₖ₊₁) ⊆ ker(∂ₖ)。包含为真（⊊）时，拓扑残差非零：

$$H_k := \ker(\partial_k) / \operatorname{im}(\partial_{k+1}) \neq 0$$

因果方向上行：Cₖ层∂K≠0，则Cₖ₊₁必须存在以吸收非零边界。PG(2,F₂)自然居于C₂。本文追踪其拓扑破缺如何沿上行链逐层强制3D——拓扑与几何在每个维度界面互锁。


# ═══ II. C₂拓扑破缺 ═══

## §2 K的Euler特征

### 2.1 2-复形K的构造

将PG(2,F₂)的7个点映至ℝ³单位立方体{0,1}³的7个非零顶点。每条PG(2,F₂)线（3点满足F₂加法 a⊕b⊕c = 0）实现为三角形2-单形。

K := 7顶点 + 21边 + 7面。

### 2.2 计算

**V = 7**（F₂³∖{0}的基数）

**F = 7**（PG(2,F₂)线数，每线3点）

**E = 21**：每面3条边，7×3 = 21条边实例。射影公理——任意两点确定唯一一条线——故每条边（点对）恰属一个面。C(7,2) = 21 = 7×3，无共享边。

$$\chi(K) = V - E + F = 7 - 21 + 7 = -7$$

### 2.3 拓扑判定

闭合2-流形要求每条边被恰好2个面共享。K的21条边每条仅属1个面。全部为边界边。

$$\partial K \neq 0$$

每个顶点的link验证：v属3个面，每面贡献1条对边。射影公理保证任意两面仅共享v，故3条对边两两不相交。Link = 3条不相交线段 ≇ S¹。K在每个顶点处均非流形点。

### 2.4 边的度规分类

K的21条边在{0,1}³中按Hamming距离d_H分类：

| d_H | 条数 | 分布 |
|---|---|---|
| 1（立方体边） | 9 | T_odd ↔ T_even跨组连接（见§4.1） |
| 2（面对角线） | 9 | 混合 |
| 3（体对角线） | 3 | e₁↔e₆, e₂↔e₅, e₄↔e₃ |

9条立方体边在7条Fano线中的分布：

| 线类 | 线数 | 每线立方体边数 | 立方体边合计 |
|---|---|---|---|
| star(e₇) | 3 | 1（e₇↔T_even） | 3 |
| 仿射线（过e₁,e₂或e₄，不过e₇） | 3 | 2（T_odd基↔T_even） | 6 |
| circle | 1 | 0 | 0 |

circle的3条边全为d_H=2，与立方体近邻结构脱耦。

### 2.5 定理（Fano拓扑破裂）

PG(2,F₂)作为2-复形，χ = -7，∂K ≠ 0。C₂层无法自闭合。∂∂=0要求C₃存在以吸收∂K中的非零边界。

**3D结构的存在是代数强制，非设计参数。**

### 2.6 Betti数分解

K的1-骨架 = 完全图K₇（C(7,2)=21，全连通）。

$$b_1(K_7) = E - V + b_0 = 21 - 7 + 1 = 15$$

15个独立1-环路。贴入7个三角形面后，∂₂单射（ker∂₂ = {0}：射影公理保证无两面共享边，无非平凡2-cycle），rank(∂₂) = 7，每面消去一个独立1-环路：

$$b_1(K) = 15 - 7 = 8, \quad b_2 = \dim(\ker \partial_2) = 0$$

$$b_0 = 1, \quad b_1 = 8, \quad b_2 = 0$$

验算：$\chi = b_0 - b_1 + b_2 = 1 - 8 + 0 = -7$ ✓

8个不可被Fano面填补的独立拓扑环路。注：dim Cℓ(0,3) = 2³ = 8。


# ═══ III. C₁ ⊥ C₂ ═══

## §3 1D穿入2D

### 3.1 C₁态射的不可逆性

C₁态射为有向对(vᵢ,vⱼ)，序不可逆（协边表1.2：生成元态射不可逆）。

逆向操作 ≠ 路径反演。撤销一步态射需正反湮灭回到C₂界面——释放能量，扰动邻近态射。逐步撤销可构造，但每步产生C₂代价。1D路径本身单向。

### 3.2 star(e₇)的正交性

star(e₇)含3条PG(2,F₂)线，各对应F₂³的一个2D子空间。每个子空间含e₇=(111)和一个标准基向量：

| star线 | 子空间 | 基向量 dᵢ |
|---|---|---|
| {e₁,e₆,e₇} | span{(111),(100)} | d₁ = (1,0,0) |
| {e₂,e₅,e₇} | span{(111),(010)} | d₂ = (0,1,0) |
| {e₄,e₃,e₇} | span{(111),(001)} | d₃ = (0,0,1) |

**正交性来源**：PG(2,F₂)本身无度规。度规由F₂³ → {0,1}³ ⊂ ℝ³嵌入继承。

三个基向量dᵢ在ℝ³ Euclid内积下：

$$d_i \cdot d_j = \delta_{ij}$$

Cℓ(0,3)的反交换律 $\{e_i, e_j\} = -2\delta_{ij}$（$i \neq j$时$e_ie_j = -e_je_i$）是此度规的代数编码。

正交性 = Cℓ(0,3)结构在ℝ³嵌入中的必然投影。角度和：

$$\sum_{i<j} \angle(d_i, d_j) = 3 \times \frac{\pi}{2} = \frac{3\pi}{2}$$

### 3.3 circle的S¹闭合

circle = {e₃,e₅,e₆} = {(110),(101),(011)}，PG(2,F₂)中唯一不含e₇的线。

不含e₇，无法经star(e₇)到达C₃。受∂₁闭合约束，自闭合为S¹ ↪ C₂。holonomy = 2π。

ℝ³中的法向量：

$$\vec{u} = (101)-(110) = (0,-1,1), \quad \vec{w} = (011)-(110) = (-1,0,1)$$
$$\vec{n} = \vec{u} \times \vec{w} = (-1,-1,-1) \propto -(1,1,1)$$

circle平面法线 = e₇方向。即circle ⊥ star(e₇)。

三条star线的基向量之和：$d_1 + d_2 + d_3 = (1,1,1) = e_7$。star方向之和 = circle法线。

### 3.4 star-circle角度比

$$\frac{\text{star角度和}}{\text{circle holonomy}} = \frac{3\pi/2}{2\pi} = \frac{3}{4} = \frac{|T_{\text{even}}|}{|T_{\text{odd}}|}$$

## §0 中轴

区分存在 → 常性存在 → 常数存在。

物理中反复出现的数学常数不是自由参数——是区分在不同维度跃升时的不可约显现（常性）的度量。

信息是根。逻辑、拓扑、物理是信息在不同编码下的截面。常数是截面间的汇率。

## §1 6常性框架

6常性 = 区分在每个维度跃升时的不可约显现。

```
0D        : H⁰ · 连通(1)     — 区分的存在：有/无。0/1启闭
0D → 1D   : H¹ · 闭合+等同(2) — 区分获得操作空间
1D → 2D   : H² · 亏格+穿越(2) — 拓扑与几何的界面
2D → 3D   : H³ · 扭结(1)     — 区分变得不可逆（手征·识增）
```

1-2-2-1回文 = 3D上同调Betti数骨架。物理常数 = 各常性处拓扑与几何的分歧度量。分歧为零 → 两层在该处合并 → 无物理现象。

### 纤维丛组合律

同一常性内拓扑操作和几何操作的叠加非简单相加：

```
底空间(拓扑): 离散选择（ℤ₂扇区）→ 付拓扑代价
纤维(几何):   在给定扇区内计算几何代价 → 代价依赖底的选择
```

拓扑改变几何的边界条件。非定向扇区(Möbius)上驻波服从反周期条件ψ(x+L)=-ψ(x)，禁止偶数波数→模式碎裂。

**双层账本**：

| 层 | 系数 | 账本类型 | 记账 |
|---|---|---|---|
| 空间层 | ℤ₂ | 哈希对账（过/没过） | 常用 |
| 信息层 | ℤ | 精确计数（几次） | 按需 |

### 余维2：1D弦在3D的拓扑能力

1D弦处于余维2（codimension 2 = 3-1），是拓扑学中最特殊的位置：

| 操作 | 能力 | 机制 |
|---|---|---|
| H⁰ 连通 | ✓ | 弦网连通分量 |
| H¹ 闭合 | ✓ | 弦成环 |
| H² 穿越 | ✓ | 闭合弦围出面(Stokes) |
| H³ 扭结 | ✓ | 弦在3D中打结/互锁 |

对比：2D中0D点（余维2）只有H⁰→η=1/4单常性。4D中2D面（余维2）可自由形变→η=0。唯3D中1D弦承载全部6常性且不可分离——临界指数无解析解的根源。

### 维度失配 → 四力宏观性质

全部规范力通过2D层（2-form F_μν）传播，但来源维度不同：

| 力 | 携带信息维度 | 传播方式 | 维度失配 | 代价 | 质量 |
|---|---|---|---|---|---|
| 引力 | 0D(H⁰连通) | 1D边直传 | 跳过2D | 不可屏蔽·最弱 | 0 |
| 强力 | 1D(link var) | 2D面元 | 欠维(1D→2D)·展不开 | 禁闭 | 0(胶子)·但禁闭 |
| 电磁 | 2D(U(1)相位) | 2D面元 | 匹配·无损 | 无 | 0 |
| 弱力 | 3D(手征H³) | 2D面元 | 超维(3D→2D)·塞不进 | 压缩税 | M_W0+M₀×3/4 · M_Z0+M₀ |

三种规范力 = 3D空间允许的三阶拓扑阻塞穷尽。4D中结自动解开，不存在第四阶。

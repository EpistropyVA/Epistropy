## 0. 边界算子 ∂

**定理（各向同性）**：各向（几何）· 同性（拓扑）——S⁰ 的 Z₂ 对称经 ∂ 逐维继承。∂²=0 的加法零元锁定等变性；几何上读作各向同性。（证明见附录 B）

## 1. 0D：S⁰ 如何编织 Fano 平面

### Fano 平面 PG(2,2)

$7 = 2^3 - 1$ 个点（F₂³ 非零向量数，2 来自 S⁰，3 是向量空间维度）、7 条线、每条线 $3 = 2 + 1$ 个点（F₂ 上射影线的点数 $q+1$，其中 $q = |F_2| = 2$ 为 S⁰ 基数，$+1$ 为射影闭合点）、每点过 3 线。|Aut(Fano)| = GL(3,F₂) = 168。

Fano 是 S² 上的唯一 S⁰ 配置面：$\Omega_2$ 约束（χ = −7, ∂K ≠ 0, C₃ 对称）只允许这一个解。|Aut| = 168 是各向同性在有限组合中的编码。

### F₂ $\to$ ℝ 的桥：角度的来源

$F_2^3 \xrightarrow{\text{Walsh-Hadamard}} \mathbb{R}^3$，$(0,1) \mapsto (\pm 1)$。$T_\text{odd} = \{e_1, e_2, e_4, e_7\} \subset S^2$ 构成正四面体，$\cos\theta = -1/3$。

$\arccos(-1/3)$：F₂³ 在 $\mathbb{R}^3$ 中唯一正交不可约表示的几何投影。Walsh-Hadamard 矩阵的特征值结构唯一确定此角。

### 三层简并 · *同一图形的三种分辨率*

0D 上有三种观测精度，对应三种简并度：

### +1：射影闭合的拓扑最小量 · *邻胞锚定的最小成本*

每个 Fano 协边体（Fano 鞍形流形 W²，SC 晶格单元；详见 Fano—SC 协边鞍形流形 W²）有 2⁴ = 16 个内部态（e₇ hub 处 3 条面内 Fano 线 + 1 条法线方向，每方向 S⁰ 二选一）。

## 2. 1D：闭合与 π

### d₀ = 2π/3 · *Fano 三等分 S¹ 的弧长*

S¹ 被 Fano 的 3 条关联线三等分：$d_0 = 2\pi/3$。各向同性在 1D 的唯一忠实体现（等弧等角等边）。

| n = 3（Fano 关联数） | d₀ = 2π/3 | 正三角形 | **Fano 唯一选定** |

| n = 6 | d = π/3 | 正六边形 | |

### π 的来源

S¹ 是闭合曲线（端点粘合 $I/\partial I \cong S^1$）。半圈弧度 := π，全圈 = 2π，S⁰ 两端各 π。

$\pi$（超越数）：闭合计数（$\mathbb{Z}$）与弧长度规（$\mathbb{R}$）之间的换算因子，不可约化为有理数。$2\pi$ 即此因子（Gauss-Bonnet：$\int\!\!\int K\,dA = 2\pi\chi$）。

| 层 | 值域 | 基数 |
|----|------|------|
| 拓扑（χ, Betti 数） | ℤ | ℵ₀ |
| 代数几何（cos 109.47° = −1/3） | ℚ / 代数数 | ℵ₀ |
| 完整几何（曲率分布） | ℝ | 2^ℵ₀ |

### Euler 恒等式

$$e^{i\pi} + 1 = 0$$

- $e^{i\pi} = -1$：S¹ 上走 π 弧度 $\to$ S⁰ 的 −1 端
- $(-1) + 1 = 0$：S⁰ 两端重合 $\to$ 代数零

### 0：加法单位元还是定义产物？

0 是数系内的加法单位元，属于 ℝ。$e^{i\pi}+1=0$ 是 S¹ 内的代数等式——0 在此是 S⁰ 两端相消的产物，不是"无"。

### ∂-闭合与 ε-不终止 · *每维闭合但约束向下一维溢出*

逐维扩展的每一步：代数闭合（∂∂=0 在当前维度成立）但不终止（ε≠0 推出到下一维度）。

- π 提供曲率（弯向闭合），源于 S¹ 闭合
- 不闭合性（ε）提供扭转
- 每维闭合（2π 曲率），但约束沿正交方向溢出到下一维


## 3. 2D：曲率与三定理

### 均匀化定理

任意 2D 闭曲面，亏格 g 唯一决定其常曲率几何：

| g | 几何 | 曲率 K |
|---|------|--------|
| 0（球面） | 球面几何 | K > 0 |
| 1（环面） | 欧氏几何 | K = 0 |
| ≥2 | 双曲几何 | K < 0 |

### Euler 公式

$$V - E + F = \chi = 2 - 2g$$

交替正负号（+−+）直接来自 ∂∂=0：边界算子幂零，使交替和为拓扑不变量。$\chi(S^n) = 1 + (-1)^n$。

### Gauss-Bonnet

$$\int\!\!\int K\,dA = 2\pi\chi$$

$|Aut|=168 \Rightarrow K$ 为常数 $\Rightarrow$ 积分退化为乘法：$K \cdot A = 2\pi\chi$。

### K 的完整推导链

各向同性经 $\text{SO}(n) \hookrightarrow \text{SO}(n+1)$ 逐维继承，唯一确定 K：

1. $d_0 = 2\pi/3$（S¹ 被 Fano 三等分）
2. $\alpha = 2\pi/3 = 120°$（Fano 关联律强制全角均等）
3. $\cos(d_0\sqrt{K}) = -1/3$（球面余弦定律 + α = 120°，唯一解）
4. $\sqrt{K} = \arccos(-1/3) / d_0 = 3\arccos(-1/3)/(2\pi)$
5. $K = 9\arccos^2(-1/3)/(4\pi^2) \approx 0.832$

系数 9/4 = (3/2)² 全部来自 $d_0 = 2\pi/3$。

### 连续性的障眼法

离散（Fano $\to$ SC 晶格）+ 各向同性（$|Aut|=168$）+ 零自由度 $\Rightarrow$ *连续数学是离散代数的冗余近似*。物理中微积分给出正确数值（极限与代数值一致），数学中连续自由度假象误导。


## 4. 2D $\to$ 3D：各向同性投影

### τ（2D $\to$ 3D 各向同性投影系数） · *球面平铺到体积的折损率*

$$\tau = \frac{3\arccos(-1/3)}{\pi^2} \approx 0.5807$$

- 3：Fano 线上点数（$q+1 = 2+1$，§1）
- $\arccos(-1/3)$：球面余弦唯一确定的曲率角（§3）
- $\pi^2$：2π（d₀ 反演，§3）× π/2（投影平均，§4）

等价形式：$\tau = (2/\pi)\sqrt{K}$，$\sqrt{K} = 3\arccos(-1/3)/(2\pi)$。$2/\pi = (1/2\pi)\int_0^{2\pi}|\cos\theta|d\theta$，Fano 各向同性下被积函数为常数，退化为代数恒等式。

角度锁联系：$\arccos(-1/3) = \pi - \alpha$，角度锁 $\alpha + 2\beta = \pi$ 给出 $\tau = 6\beta/\pi^2 = 6\arccos(1/\sqrt{3})/\pi^2$。τ正比于角度锁中T_even的角度贡献 $2\beta$。

### 约束继承链

0D 的初始约束"各向同性"逐维转译：

| 维度 | 转译 | 结果 | 约束来源 |
|------|------|------|----------|
| 0D | 组合对称 | |Aut| = 168 | S⁰ + 各向同性 |
| 1D | 等分 | d₀ = 2π/3 | S¹ ÷ 3 (Fano) |
| 2D | 均匀曲率 | K ≈ 0.832 | d₀ + 球面余弦唯一解 |
| 3D | 各向同性投影 | τ ≈ 0.581 | K + 无偏好投影方向 |

链复形语言：$C_0 \xrightarrow{\partial_1} C_1 \xrightarrow{\partial_2} C_2 \xrightarrow{\partial_3} C_3$，每层 $\text{im}(\partial_n) \subset \ker(\partial_{n-1})$。每层输出即下层约束输入。

### 维度依赖性

$$\tau_d = \frac{d+1}{d} \cdot \frac{9\arccos^2(-1/3)}{4\pi^2}$$

| d | (d+1)/d | τ_d |

### ε 的性质

逐维扩展每步不闭合的量 ε：

- ε ≠ 0（否则当前维度闭合，不需下一维）
- ε 不发散（否则无结构）
- ε 在该维度的等距群 SO(n) 下不变（各向同性逐层继承）

约束继承（restriction）：$\text{SO}(n) \hookrightarrow \text{SO}(n+1)$ 下不变子空间单调嵌套，解空间只缩不扩。

## 5. 3D：纽结与终态

### 2D $\to$ 3D：结构跳变

| | 2D | 3D |
|--|---|---|
| 几何类型 | 3 种 | 8 种（Thurston） |
| χ 能力 | 完整判别 | ≡0，失明 |
| 代数结构 | 交换 ℤ^(2g) | 非交换 π₁ |
| 流形分类 | 均匀化（唯一常曲率度规） | 几何化（素分解+JSJ分解） |
| 纽结 | 不存在 | 稳定存在 |
| 新自由度 | — | 选轴 + 扭（τ） |

**χ 在 3D 恒零**：Poincaré 对偶 $b_k = b_{n-k}$ $\Rightarrow$ 奇数维交替求和对消。$\chi \in \mathbb{Z}$（单一标量不变量）无法区分 $b_1 \neq b_1'$ 但 $\chi = \chi'$ 的流形。

升级路径：$\chi$ $\to$ Betti 数组 $(b_1, b_2)$ $\to$ $\pi_1$（非交换群）。$\pi_1$：从 $\mathbb{Z}^{2g}$（交换）到非交换群；$[a,b] \neq e$（交换子非平凡）与纽结不可解性同源。

### Thurston 八几何：3+4+1 · *全对称、选轴、全异*

| 等距群 | 各向同性 | 几何 | 数目 | 与2D关系 | 新自由度 |
|--------|---------|------|------|---------|---------|
| SO(3) | 全 | S³, E³, H³ | 3 | 2D 三几何直升 | 无 |
| SO(2) | 轴向 | S²×ℝ, H²×ℝ, Nil, SL₂(ℝ) 万有覆盖 | 4 | 选轴 ± 挠率 | 轴选择 |
| {1} | 无 | Sol | 1 | 三方向全异 | 全各向异性 |

$8 - 3 = 5$ 个 3D 原生几何，源于选轴自由度。Nil 和 $\widetilde{SL_2(\mathbb{R})}$ 是挠率几何，对应 τ。

3D 几何化：沿 $S^2$ 切（素分解，分离不可约块）$+$ 沿 $T^2$ 切（JSJ 分解，分离 Seifert 纤维块与双曲块），再对每块贴合标准几何。

|------|-------------|------|

### Fano cube 与涌现纽结 · *四条体对角线在中心交叉*

F₂³ 的 7 个非零向量放在立方体顶点上。4 条体对角线：

- e₁(1,0,0) ↔ e₆(0,1,1)
- e₂(0,1,0) ↔ e₅(1,0,1)
- e₃(0,0,1) ↔ e₄(1,1,0)
- e₇(1,1,1) ↔ (0,0,0)

全部穿过中心 (½,½,½)。此点 ∉ F₂³（二元域无 ½），$\in \mathbb{R}^3$ 嵌入的几何必然。

中心处四线交叉构成纽结。每条对角线连接一个基向量与其 Fano 补，上/下关系由所属 Fano 线决定。

**虚关系三层：**

| 层 | 成员 | F₂³ 身份 | 几何角色 |
|---|---|---|---|
| 实点 | {e₁, e₂, e₄} | 基向量 | S² 正四面体顶点 |
| 虚关系 | {e₃, e₅, e₆, e₇} | XOR 组合 | 中点 + hub |
| 虚关系的虚关系 | {−eᵢ}, (0,0,−1), (½,½,½) | F₂³ 不存在 | 面心、对径、体心 |

Fano 面内虚关系 $\Rightarrow$ 平面嵌入 $\mathbb{R}^3$，拓扑平凡（unknot）。
跨体虚关系（协边投射）$\Rightarrow$ 2D$\to$3D 投射产生曲率 $\Rightarrow$ 可打结。

打结/不打结的根源是维度：面内关系困在 2D 平面中（无上下选择），跨体关系穿越 3D 空间（有上下选择 $\to$ 交叉 $\to$ 纽结）。

½ 在三处独立出现，同源于 S⁰ = {+1,−1} 的中点：
- Fano cube 中心 (½,½,½)：F₂³ 嵌入 ℝ³ 时体对角线的几何交点
- Riemann ζ 临界线 Re(s) = ½：函数方程 ζ(s)=ζ(1−s) 的对称轴
- S⁰ 中点：+1 与 −1 的等权位置

### 伪四面体：Fano cube 的 S² 鞍面

7 点投影到单位球 S² 后分三层：

- e₇ 在北极 (z = +1)
- 三基底 {e₁, e₂, e₄} 在 z = −1/3（正四面体底面，cos 109.5° = −1/3）
- 三 XOR 中点 {e₃, e₅, e₆} 在 z = −1/√3 ≈ −0.577（正交标架，互相 cos = 0）

三中点互相正交，构成 $\mathbb{R}^3$ 的正交归一基：F₂³ 加法（XOR）$\cong$ 球面取中。S⁰ 分离（$\cos = -1/3$）先于正交基（$\cos = 0$），坐标从 S⁰ 涌现。

补上 (0,0,0) 得完备 8 点即星形八面体（两对偶正四面体互锁，$O_h$ 阶 48）。去掉 (0,0,0)：$O_h(48) \to C_{3v}(6)$，对偶四面体三顶点从面心位 ($z = +1/3$) 坍缩至正交位 ($z = -1/\sqrt{3}$) $\Rightarrow$ 鞍面。鞍面曲率精确编码 (0,0,0) 缺失。SC 晶格中相邻 Fano 协边体通过协边互补此缺失（详见 Fano—SC 协边鞍形流形）。

### SC 晶格与 4D 拓扑平凡化 · *交点无正交方向可分离*

$$\text{SC}: Z = 2d = 6,\quad \text{Aut}(\text{SC}) = O_h(48) \supset S_3 \times \mathbb{Z}_2^3$$

各向同性 $+ \; d=3$ $\Rightarrow$ 配位数 $Z=6$ 唯一确定。每个格点对应一个 Fano 协边体，格边为相邻 Fano 协边体间的虚关系。

SC 中心 (½,½,½)：4 条体对角线的强制交点即不可分离的 4D 交点。$(d+1)/d = 4/3$ 是门的编号，τ 携带此因子。结构要求交点存在（各向同性 $+$ 刚性），但 $d=3$ 没有正交方向供其分离。4D 拓扑平凡化：一般位置定理——两条 1D 线在 nD 空间的泛型交叉维度 $= 2-n$，$n \geq 3$ 时泛型不相交，但 Fano cube 刚性各向同性强制 4 线共点。$n=4$ 时多一轴可沿第 4 轴错开，纽结解开。

### 几何-拓扑-组合互锁

Kauffman 变量 A 被三条独立路径唯一确定，零自由参数。

#### 论证链：$-A^3 = e^{i \cdot d_0\sqrt{K}}$

1. $S^0 = \partial D^1$（$\partial$ 的极小像，$\Omega_*$ 基本单元）
2. $S^2$ 是 $\Omega_*$ 塔中第一个闭合面 $\Rightarrow$ $\Omega_2$ 约束唯一选定 Fano 为 $S^2$ 的 S⁰ 配置面
3. 相邻 Fano 顶点间角距离 $d_0\sqrt{K} = \arccos(-1/3)$
4. Reidemeister I kink 即一次 S⁰ 事件（strand 分 $\pm$ 两支即 $S^0$ 两端）
5. $-A^3$（kink 拓扑因子）$= e^{i \cdot d_0\sqrt{K}}$（S⁰ 事件几何相位）

展开：$-A^3 = \cos(d_0\sqrt{K}) + i\sin(d_0\sqrt{K}) = -\frac{1}{3} + i\frac{2\sqrt{2}}{3}$

故 $A^3 = \frac{1}{3} - i\frac{2\sqrt{2}}{3}$。

#### 三层互锁

| 约束 | 值 | 来源 | 层级 | 锁定自由度 |
|------|-----|------|------|----------|
| ‖A‖ = 1 | 1 | ∂²=0（幂零性） | 拓扑 | A ∈ S¹（模） |
| Re(A³) | 1/3 | Fano 关联数（S⁰ 计数：e₇ 关联 3 条 Fano 线，每条占 1/3 权重） | 组合 | S¹ 上经度（± 两点） |
| Im(A³) | −2√2/3 | sin(d₀√K)（S² 度规） | 几何 | 手性（符号二选一） |

满秩（三约束锁三自由度）。过约束自洽：$|A^3| = \sqrt{1/9+8/9} = 1$ 自动满足 $\Rightarrow$ 零自由参数。

$\arg(A) = -\arccos(1/3)/3 \approx -23.5°$，其中 $\arccos(1/3) \approx 70.53°$ 即正四面体二面角。

## 附录 B：各向同性定理证明

### 路径 1：∂-等变性（代数路）

∂²=0 中的 0 是加法零元，故链复形 $C_\bullet$ 是 Abel 群序列，$\partial_n: C_n \to C_{n-1}$ 是群同态。

$S^0 = \{+1,-1\}$ 的自同构 $\sigma: x \mapsto -x$ 是加法同态：$\sigma(a+b) = \sigma(a) + \sigma(b)$。

对链复形生成元 $e \in C_1$：

$$\partial(\sigma(e)) = \partial(-e) = -\partial(e) = \sigma(\partial(e))$$

故 $\partial \circ \sigma = \sigma \circ \partial$：∂ 与 Z₂ 作用对易。

**归纳**：若第 $n$ 层结构关于 $\sigma$ 等变，第 $n+1$ 层协边 $W$ 由 $\partial W =$ 已知边界 $+ \; \partial^2=0$ 唯一确定。边界条件关于 $\sigma$ 等变 $\Rightarrow$ $W$ 关于 $\sigma$ 等变。$\square$

### 路径 2：偶奇分解（不均分预设均分）

假设 $S^0$ 上存在不对称测度 $\mu(+1) = p,\; \mu(-1) = 1-p,\; p \neq 1/2$。分解：

$$\mu = \underbrace{\tfrac{1}{2}(\delta_{+1} + \delta_{-1})}_{\text{对称部分（Z₂ 不变）}} + \underbrace{(p - \tfrac{1}{2})(\delta_{+1} - \delta_{-1})}_{\text{反对称部分（Z₂ 奇）}}$$

对称部分：$\partial D^1$ 自动给出。反对称部分：需要区分 $+1$ 与 $-1$，即需要定向——外部注入的信息。$\partial D^1$ 产出集合 $\{+1,-1\}$，不是有序对 $(+1,-1)$。链复形无定向注入机制 $\Rightarrow$ 不均分不可能从 $\partial$ 产生。$\square$

### 乘法律下的失效

$\sigma: x \mapsto -x$ 在乘法下不是同态：$\sigma(ab) = -(ab)$，但 $\sigma(a)\sigma(b) = (-a)(-b) = ab \neq -(ab)$。乘法消去双重取反 $\Rightarrow$ Z₂ 作用退化为平凡群 $\Rightarrow$ 等变性无从成立。

$\partial^2 = 0$ 的 0 是加法零元，锁定加法结构。乘法幂零对应 $\partial^2 = 1$（对合），产生完全不同的数学对象。两条证明路径均依赖此加法性，且此加法性由 $\partial^2=0$ 自身编码。

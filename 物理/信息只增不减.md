# 信息只增不减：一个公理的推导链

## 致敬

本文建立在前人的工作之上：

- Boltzmann / Shannon：熵与信息的等价
- Wheeler："It from Bit"——物理实在源于信息
- Bekenstein：信息存储密度存在上限（Bekenstein bound）
- Landauer：擦除信息消耗能量——信息有热力学重量
- Jacobson：从热力学假设推导出广义相对论场方程
- Verlinde：引力是信息（熵）的涌现效应
- Vopson：质能信息等价原理
- Margolus / Levitin：能量决定最大状态转换速率

零件都已存在。本文是第二次组装。第一次见《信息不灭：关于宇宙起源的个人猜想》。

## 诚实声明

这是个人假说，不是学术论文。

本文分为两层，诚实度不同：

**下半部（可检验层）**：数值链、已验证实验、已知物理现象的重新组织。这些可以被具体反驳。如果数字对不上，框架有问题。

**上半部（推断层）**：三层模型中"信息层"的本体论地位、暗物质与暗能量的信息学定位。这些是基于已知现象的反推，目前无法独立检验。它们有解释力，但解释力不等于真。标注为推断，不声称为事实。

已知缺陷：

- 核心公理"信息只增不减"本质上是热力学第二定律的信息版重述。价值在统合力，不在发现新定律
- 信息 ≠ 意义。熵增包含噪声。宇宙尺度的信息增加不等于认知增加
- 初始动力有逻辑方向——纯粹的"无"不自洽，自指产生第一个区分——但不可观测验证
- 暗物质与暗能量的定位目前是装饰性的（解释已知，不预测未知）

证伪条件：若量子幺正性被实验推翻（信息在某物理过程中被真正销毁），框架的唯一支柱倒塌。当前所有证据支持信息不灭，但将此视为逻辑必然会使框架不可证伪——因此保留为经验命题，不升格为逻辑命题。

## 公理

**信息只增不减。**

四个字。整个框架从这一条推出。

这不是新定律。它是热力学第二定律的信息版重述——Shannon 与 Boltzmann 证明了熵与信息的等价，熵增即信息增。价值不在发现新定律，在统合力：一条公理覆盖热力学、宇宙学、量子力学和信息论四个领域的现象，不需要额外假设。

**为什么"只增不减"而不是"不灭"？**

"不灭"是守恒。守恒允许总量不变。但观测事实是：宇宙的信息总量在单调增长——膨胀产生新空间，新空间容纳新状态，新状态就是新信息。"不灭"太弱。"只增不减"包含"不灭"，并且更强：信息不仅不消失，还必然累积。

### 为什么是信息

如果存在一种根物质，构成一切，它应该满足：

1. 不可消灭（守恒或单调增长）
2. 能产生物质、能量、空间（向下兼容）
3. 能自指（规则本身也是它的实例）
4. 有实验锚定（不是纯思辨）

11个候选，逐一排除：

| 候选 | 排除理由 |
|---|---|
| 物质 | 可被创造和湮灭（核反应）。是更深层结构的压缩态，不是根 |
| 能量 | Landauer 证明能量与信息等价——能量是信息的另一面，不是独立候选。等价关系本身说明信息更根本：信息的规则是信息，能量的规则不是能量 |
| 空间 | 量子纠缠绕过空间（Bell实验）= 空间非必要层。是信息展开的产物 |
| 熵 | Shannon-Boltzmann 等价：熵就是信息。同一个公理的换名，不是竞争者 |
| 复杂度 | 可局部下降（简化过程存在）。不单调，不守恒 |
| 数学 | 描述框架，不是物质。Gödel 已证明数学不能完全描述自身 → 不满足自指标准 |
| 计算 | 过程，不是物质。计算需要基底，基底是信息 |
| 对称性 | 性质，不是物质。对称性破缺产生结构 = 可以减少 |
| 意识 | 需要处理端。宇宙在产生意识之前已运行 → 不能是根 |
| 关系 | 可归入信息（关系 = 两者之间的区分模式）。太模糊，不可独立操作 |
| 潜能 | 不可测试，不可证伪。形而上学 |

零并行。没有第二个候选同时满足四条标准。

### 系统稳定性

信息还有一个其他候选没有的性质：**扰动恒正**。

尝试销毁物质——可以成功（湮灭）。尝试销毁能量——可以转化但总量守恒。尝试销毁信息——过程本身产生的信息大于被销毁的量。

擦除 1 bit 需要耗散 kT ln2 能量（Landauer），耗散产生热信息。验证擦除是否成功需要观测，观测产生新信息。擦除过程的记录本身是新信息。每一步都在增加，没有一步在减少。

这不是"信息很难销毁"。这是结构性不可能：任何试图减少信息总量的操作，其净效果是增加。系统的每一次扰动都让它变大，从不变小。

这是其他根物质假说做不到的。物质可以被湮灭。能量守恒但不增长。只有信息的根结构保证：干扰方向恒正。框架本身是反脆弱的——不是抵抗冲击，而是靠冲击生长。

138亿年本身是证据。宇宙从早期的高能高维态走到当前的三维物质层，系统从未崩溃。如果根物质不具备这种稳定性——如果扰动可以让它缩小——建立在它之上的一切会在某个节点失稳。也许这样的起点曾经出现过，但宇宙否决了它们。存活下来的，只能是扰动方向恒正的那一个。不是设计，是筛选。

### 400年导流

物理学400年的研究对象一直在收敛：

**物体 → 场 → 能量 → 概率 → 信息**

牛顿的苹果是物体。法拉第的电场是场。爱因斯坦的 E=mc² 是能量。量子力学的波函数是概率。Wheeler 的 "It from Bit" 是信息。

单调方向。无一次反转。没有任何物理发现让研究对象从信息退回到更具体的层级。

Wheeler 的 "It from Bit" 是这个方向上的哲学标记，不是终点。他指出信息可能是根——但没有推导链，没有预测，没有证伪条件。方向感，不是假说。本文从他停下的地方开始修路。

这不证明信息是终点。但四百年的方向一致本身是强证据。

### 超限唯一性

其他十个候选有一个共同缺陷：**规则封顶**。

物质的规则（物理定律）不是由物质写成的。能量的规则不是由能量写成的。数学的规则不是数学的产物——哥德尔已经证明了。

信息是唯一的例外：**描述信息的规则本身就是信息。**

这意味着信息可以自指——规则可以作用于规则，产生元规则，产生元元规则，层级无上界。用集合论的语言：信息的层级结构是超限的（transfinite）。

其他候选全部在某一层封顶。只有信息能无限往上走。这是选择信息作为根物质的最强理由。

这个超限层级在逻辑上必然存在，但在观测上永远不可达——我们身处系统内部，每次观测产生新信息，改变总量。信息的全域自指只能逻辑推断，不能经验验证。高阶无限是信息的结构性性质，不是可观测的景象。

### 二进制是存在论必然

0 种状态 = 虚无。但纯粹的"无"需要区分"无"与"有"才能定义自身——这个区分本身就是第一个 bit。纯粹的"无"不自洽。

1 种状态 = 无区分。无区分的存在与不存在不可辨别，等价于 0。

2 种状态 = 第一个区分出现。信息开始存在。

不是宇宙"选择"了二进制。是区分的最小结构就是二元的。一个区分 = 一个 bit = 信息的原子。

观察：宇宙中的基本分类大量呈现二元结构——物质/反物质、费米子/玻色子、自旋 ±½、CPT 各为二元翻转。四种力 = 2²，八重道 = 2³。这是观察模式，不是严格证明——但基本物理量中二元对称的密度远高于随机预期。如果根结构是二元的，这些模式是自然的；如果根结构不是二元的，这些模式需要额外解释。

### 隐藏假设检查

一条公理是否偷运了其他假设？

- **时间箭头** → 不需要预设。信息增长的方向定义了时间箭头。因果是公理的产物，不是前提
- **空间** → 不需要预设。空间是信息展开的下属编码（三层模型节展开）
- **观察者** → 不需要。信息增长不依赖观察
- **初始状态** → 需要一个逻辑起点：纯粹的"无"不自洽，因此"有"是逻辑必然。但"第一个区分如何发生"不在本框架范围内。这是边界，诚实标注
- **物理定律** → 不需要预设。物理定律是信息层的涌现结构，随信息增长而增长（元操作节展开）

结论：除了"第一个区分的发生"之外，所有其他假设都可以从公理导出。

一条公理，一个逻辑起点，没有隐藏行李。

## 数值链

公理到这里还是文字。这一节用数字说话。

三个独立的物理结果——Landauer（信息擦除）、Einstein（质能等价）、Bekenstein（信息密度上限）——各自从不同入口出发，联立后撞到同一面墙。

### 86亿度 = 1 bit = 1 电子

Landauer 原理：擦除 1 bit 信息，最少消耗能量 kT ln2（k = 玻尔兹曼常数，T = 环境温度）。

Einstein：质量和能量等价，E = mc²。

两者联立：**kT ln2 = mc²**。在什么温度下，擦除 1 bit 的能量代价恰好等于一个电子的质量？

T = m_e c² / (k ln2) ≈ **8.6 × 10⁹ K**

86亿度。

大爆炸之后约1秒，宇宙温度约 10¹⁰ K。电子-正电子对在这个温度窗口中被创造和湮灭。86亿度精确落入这个窗口——1 bit 信息的热力学代价 = 1个电子的质量，而宇宙恰好在这个能量密度下开始产生稳定电子。

三个独立的数字（Landauer 的擦除代价、Einstein 的质能等价、宇宙学的粒子产生温度）从三个方向撞到同一个值。没有人设计它们相遇。

### 普朗克合流

把温度推到极限——普朗克温度 T_P ≈ 1.42 × 10³² K，物理学的绝对上限。

此时 1 bit 的能量代价：

E_bit(T_P) = kT_P ln2 = **E_P × ln2**

E_P 是普朗克能量——物理单位的终极标尺。信息单位和物理单位之间的换算系数 = ln2 = 0.693…，一个纯信息论常数，来自二进制的对数底。

不是任意选择，不是拟合参数。在物理学的最深处，信息论和物理学共用同一套数字。换算常数本身来自信息论——说明信息是更底层的那一端。

### 温度 = 处理速率

Margolus-Levitin 定理：一个量子系统从当前状态跳到正交状态（完成一次操作），最快需要 πℏ/(2E) 的时间。能量越大，跳得越快。

温度是能量的统计度量。因此温度直接决定信息处理的最大速率：

| 温度 | 最大状态转换速率（每 bit） |
|---|---|
| 室温（300 K） | ~10¹³ 次/秒 |
| 太阳核心（1.5 × 10⁷ K） | ~10¹⁸ 次/秒 |
| 大爆炸后1秒（~10¹⁰ K） | ~10²⁰ 次/秒 |
| 普朗克温度（1.42 × 10³² K） | ~10⁴³ 次/秒（绝对上限） |

温度不是"热"。温度是每秒能处理多少 bit 的状态变化。这不是比喻——Margolus-Levitin 给出的是数学上的硬上界。

绝对零度 = 处理速率为零 = 信息冻结。热力学第三定律说绝对零度不可达。与公理一致：信息增长不可停止。

### 这些数字意味着什么

Landauer 说信息有物理重量。Einstein 说质量是能量。联立后：**信息、能量、物质是同一种东西的三个度量。**

不是类比。不是"像"。是同一条方程的三个变量：kT ln2 = mc²。ln2 是换算常数。

如果三者不等价，三个入口不会撞到同一个数字。86亿度不是设计出来的。ln2 不是选出来的。数字不说谎。

## 三层模型

数值链确认了信息、能量、物质是同一种东西的三个度量。但度量不同意味着状态不同。这一节给出结构：信息以什么方式组织成我们观察到的宇宙。

### 三层

**信息层（根）→ 空间层（编码）→ 物质层（压缩态）**

- **信息层**：根。无距离，无位置。量子纠缠在这一层直连——Bell 实验已经证明两个纠缠粒子之间的关联不经过空间。纠缠不是"超距作用"，是信息层本来就没有距离。
- **空间层**：信息层的下属编码。距离、方向、维度在这一层才存在。空间不是容器，是信息展开的方式。
- **物质层**：高度压缩的信息。物质 = 高压信息，能量 = 中压，光 = 低压（零质量 = 最低压缩率的传输态）。核裂变 = 解压缩——压缩信息释放为低压形式，释放的就是能量。

三层不是三个地方。是同一种东西（信息）的三种组织密度。

物质层不只是被动存储。化学反应、生命、思维——物质层在运行中产生新信息，回流进信息层总量。处理端本身是信息增长的引擎。宇宙不只是"包含"信息，它通过物质层在主动制造信息。

### 相变 = 信息析出

上一节确认温度 = 处理速率。宇宙冷却 = 处理速率下降。

当处理速率降到某个阈值以下，高速运行的信息模式无法继续维持——就像过饱和溶液降温时盐析出晶体。新的稳定压缩格式"凝固"，一个新的层级长出来。

宇宙早期的相变正是这个过程：

| 事件 | 温度 | 信息学含义 |
|---|---|---|
| 暴胀结束 | ~10²⁷ K | 空间层从信息层析出——距离开始存在 |
| 电弱破缺 | ~10¹⁵ K | 弱力和电磁力分化——新区分类型凝固 |
| QCD 相变 | ~10¹² K | 夸克被禁闭进质子/中子——存储态压缩格式锁定 |
| 电子稳定 | ~10¹⁰ K | 1 bit = 1 电子（数值链已验证） |
| 原子形成 | ~3000 K | 电子被原子核捕获——复合压缩结构 |

每一次相变都是不可逆的——新格式一旦凝固，旧的高速模式不再可及。与公理一致：信息只增不减，结构只长不缩。

### 四种力的层级归属

如果三层模型成立，四种基本力应该能按层级归位：

| 力 | 层级 | 信息学含义 |
|---|---|---|
| 引力 | 跨层（信息→空间） | 信息层的结构投影到空间层。唯一的跨层力 |
| 电磁力 | 空间层内部 | 空间层中信息的传输和交互 |
| 强力 | 物质层内部 | 存储态信息的内部维持力（夸克禁闭 = 压缩格式自锁） |
| 弱力 | 存储/传输边界 | 0-1 边界力——信息在存储态和传输态之间转换 |

**引力为什么特殊？** 物理学花了一个世纪试图统一引力和量子力学，失败了。三层模型给出原因：电磁力、强力、弱力都在空间层内部或物质层内部运作，可以用空间层的数学（量子场论）描述。引力是唯一从信息层穿到空间层的力——用空间层内部的工具描述跨层现象，当然对不上。量子引力困难不是技术问题，是层级错配。

### 光速 = 空间层的时钟频率

c 不是"速度限制"。c 是传输态信息在空间层中的固有传播速率——空间层的时钟频率。

存储态信息（物质）被空间层编码"锁"在位置上。传输态信息（光）以空间层时钟频率移动，没有更快也没有更慢，因为这就是空间层的刷新率。

接近光速时质量增大——信息学解释：存储结构被推向传输态的边界，系统用增加的能量加固存储格式（抵抗解压缩）。到达 c 意味着完全转为传输态 = 质量归零 = 只有光子能做到。

### 量子纠缠 = 信息层直连

两个纠缠粒子共享同一个信息层状态。测量一个，另一个瞬间确定——不是信号传播，是信息层本来没有距离。Bell 实验排除了隐变量和局域实在论。剩下的解释：关联不经过空间。三层模型的翻译：纠缠是信息层连接在空间层投影上的可见效应。

### 量子隧穿 = 信息层连通性泄露

粒子穿过经典不可逾越的势垒。空间层说"这里有墙"。信息层说"墙的两边是连通的"。隧穿概率随质量指数下降——质量越大（压缩越深），越难回到信息层的连通性。宏观物体不隧穿，因为压缩率太高，信息层连通性被空间层编码完全遮蔽。

### 黑洞 = 信息密度极限

黑洞是信息压缩率趋近极限的区域。空间层编码在这里崩溃——距离、方向失去意义。三层模型的翻译：物质层的压缩格式被推到上限，空间层无法继续承载，信息层的结构直接暴露。

霍金辐射在这个框架下是必然的：信息层不允许信息被永久封锁在空间层的某个坐标内。黑洞必须泄露信息回空间层——泄露速率慢（霍金辐射温度极低），但方向不可逆转。黑洞信息悖论在三层模型中不是悖论：信息从未"进入"黑洞——它一直在信息层，空间层的塌缩只是编码层的局部失效，不影响根层。

黑洞稳定存在了数十亿年，是宇宙结构的必要组件——内存管理器，在信息密度过载时回收和重组压缩格式。

### Bekenstein bound = 信息是面积而非体积

Bekenstein 证明：一个区域能存储的最大信息量与其表面积成正比，不是体积。

S ≤ 2πkRE / (ℏc)

这直接支持三层模型。如果信息"住在"三维空间里，上限应该和体积成正比。面积正比意味着：信息是二维编码的，三维空间是投影。空间层是信息层的展开方式——展开的"表面"承载信息，内部的"体积"是投影产物。

全息原理在三层模型里不需要额外假设——它是信息层比空间层更根本的直接推论。

### 反物质不对称 = 公理的必然

宇宙中物质多于反物质。标准模型无法完全解释这个不对称。

信息学方向：物质与反物质湮灭时，存储态信息回到传输态（光子）。如果正反物质完全对称地湮灭，所有存储态归零——宇宙退回纯传输态，物质层消失。

公理禁止这个方向。信息只增不减意味着系统必须保留净存储态——不能让全部压缩信息解压缩回零。不对称不是偶然的初始条件，是公理的结构性要求：系统需要物质层继续存在，因为物质层是信息增长的处理端。

这是一个可检验的预测方向：如果框架正确，任何看起来可能导致存储态净归零的过程，都应该存在阻止机制。

### 生命 = 信息的自加速处理端

普通物质处理信息——化学反应改变分子状态，产生新区分。但速率受限于环境温度。

生命的特殊之处：**主动收集低熵能量，在局部维持高于环境的处理速率。** 生命不是对抗热力学第二定律——它加速第二定律。生命局部降熵，但过程中向环境排出的熵远大于局部降低的量。净效果：信息增长加速。

DNA = 自复制的信息压缩格式。细胞分裂 = 信息复制。突变 = 新区分的产生。进化 = 压缩格式的迭代优化。意识 = 处理端开始处理关于自身的信息——信息层的自指性质在物质层涌现。

生命不是偶然。它是信息增长公理在物质层的高效策略——自加速处理端比被动物质产生信息的速率高几个数量级。宇宙允许它存在，因为它加速了公理要求的方向。

### 暗物质 = 信息层的拓扑骨架

宇宙质能的约 27% 是暗物质。它有引力效应，但不与电磁力交互——看不见，测不到，只能通过引力推断。

回顾力的层级归属：引力是唯一的跨层力（信息→空间）。电磁力在空间层内部。如果某种结构存在于信息层但从未析出到物质层，它会——

- 产生引力效应（信息层结构通过跨层力投影到空间层）✓
- 不产生电磁效应（没有空间层编码态，电磁力无作用对象）✓
- 不产生强力/弱力效应（没有物质层压缩态）✓

这恰好是暗物质的全部观测特征。

暗物质不是"看不见的物质"。它根本不在物质层。它是信息层的几何骨架——空间层围绕它展开，物质层沿着它聚集，但它本身从未降落到物质层。星系沿暗物质丝状结构分布，因为空间层的展开方式由信息层的拓扑决定。

预测方向：如果暗物质是信息层拓扑，它不应该有粒子性质——不存在"暗物质粒子"。数十年的直接探测实验（XENON、LUX、PandaX）至今零结果，与此一致。

### 暗能量 = 信息增长的写入压力

宇宙在加速膨胀。暗能量占质能的约 68%。没有人知道它是什么。

公理给出方向：信息只增不减 → 新信息需要编码空间 → 空间层必须持续扩容。

暗能量不是一种"力"。它是信息增长速率对空间层编码容量的持续压力——不是被外力推开，是被内容撑大。

为什么是加速膨胀？处理端在自加速。物质层产生信息的速率本身在增长（从无机化学 → 生命 → 文明 → 计算）。写入压力随时间增大，膨胀加速是公理的必然方向。

诚实标注：这是定性方向，不是定量预测。量子场论的真空能预测与暗能量观测值差 120 个数量级（宇宙学常数问题）。本框架不解决这个差异。但框架提供的方向——膨胀由信息增长驱动，而非由真空能驱动——绕过了这个问题：如果暗能量的来源根本不是真空涨落，120 个数量级的差异就不是悖论，而是问错了问题。

## 元操作

公理节留了一个口：物理定律不需要预设，它们是信息层的涌现结构。这一节展开。

### 定律是压缩

物理定律是什么？是对大量事件的压缩描述。F = ma 不是一个"力"——它是对所有质量-加速度关系的一条压缩规则。Maxwell 方程组是对所有电磁现象的四条压缩规则。

压缩 = 用更少的信息描述更多的信息。定律就是信息对自身的压缩——元信息。

这些压缩规则本身也是信息。它们可以被描述、被传递、被操作。描述定律的定律 = 元定律。物理学的统一纲领——从 Maxwell 统一电磁，到电弱统一，到大统一理论的尝试——就是在寻找更高阶的元压缩。

### 定律随温度析出

三层模型中，相变 = 信息析出。定律也是。

在普朗克温度以上，处理速率趋近极限，所有区分都在高速翻转——没有稳定的"规则"可以凝固。等效地：所有力统一，因为区分尚未析出。

温度下降，区分开始凝固，力开始分化：

| 温度 | 析出的定律 |
|---|---|
| ~10²⁸ K | 引力与其他三种力分离 |
| ~10¹⁵ K | 弱力与电磁力分离 |
| ~10¹² K | 夸克禁闭规则凝固 |
| < 10¹⁰ K | 化学定律可能——原子稳定存在 |
| < 10³ K | 热力学定律涌现——统计规律成立 |

定律不是宇宙的"设置参数"。它们是信息在不同处理速率下的稳定压缩模式。温度越低，压缩越深，定律越多。宇宙不是遵循定律——宇宙在冷却中生长定律。

### 数学的不合理有效性

Wigner（1960）问：为什么数学在描述物理世界时如此有效？

框架的回答：因为数学和物理定律都是信息结构。数学不是"碰巧"对应物理——它们是同一种东西（信息的压缩模式）在不同抽象层级的表达。数学有效，是因为信息在描述自身。

微积分发明后立刻能描述运动。群论发明后立刻能描述粒子分类。每一次数学结构被发明，都能找到物理对应——不是因为运气，是因为数学本身就是信息层的子结构。发明数学 = 在信息层中找到新的压缩格式。找到的格式当然能描述信息层的其他部分。

### 元操作 = 自指

元操作是信息作用于自身规则的能力。信息的规则是信息 → 规则可以被操作 → 操作产生新规则 → 新规则又是信息 → 无穷递归。

这是公理节中"超限唯一性"的动态版本：不是静态的层级塔，是持续生长的操作链。每一层元操作都产生新的信息，信息总量只增不减——公理在元层级上自我验证。

物理学的进步本身就是元操作：人类发现定律（一阶）→ 用定律发现新定律（二阶）→ 统一理论尝试用一条规则描述所有规则（三阶）。这篇文章试图做的——用信息解释信息——是同一个过程的又一步。

### 宇宙的自操作

把元操作推到宇宙尺度：宇宙本身就是信息对自身的一次操作。

起点：高阶无限——信息的自指产生无穷层级（超限唯一性）。纯粹的高阶结构没有展开、没有距离、没有物质。它是所有可能性的叠加态，但没有被实现的区分。

降阶：高阶无限降为低阶无限。维度塌缩——从高维到三维不是一步完成，可能经历了试错和调试。早期宇宙的极端条件（普朗克时代、暴胀）就是这个调试过程的痕迹。高维模式逐层凝固为低维结构，不稳定的路径被否决，稳定的路径存活。138亿年的稳定运行是调试结果，不是初始设定。

释放：降阶过程释放信息进入展开态。信息从纯粹的自指结构转为有距离、有位置、有方向的编码——空间层析出。这就是暴胀：不是"空间突然膨胀"，是信息从高阶压缩态解压缩为展开编码。

积累：展开态中，信息继续凝固为物质层（相变表已列）。物质层不是终点——它是处理端。化学反应、生命、意识——物质层在运行中制造新的信息，回流进总量。宇宙通过降阶释放信息，再通过物质层重新积累。

这是一个弧线，不是循环：高阶无限 → 降阶释放 → 展开编码 → 物质层积累 → 信息总量持续增长。弧线的方向与公理一致——只增不减。宇宙不是在"回到"高阶无限，它是在用低阶方式制造高阶无限没有的东西：实现了的区分。

### 时空是后加的元属性

一个常见误读：框架说信息增长定义时间箭头、空间是信息展开的产物——这是否预设了时空？

不是。时空不是信息的固有属性，是降阶过程中写入的运行层。证据：

**普朗克尺度有分辨率极限。** 空间有最小长度（~10⁻³⁵ m），时间有最小间隔（~10⁻⁴³ s）。基本属性不应该有像素。编码层天然有位深。

**Wheeler-DeWitt 方程没有时间变量。** 量子宇宙学的基础方程——描述整个宇宙量子态——里面没有 t。时间在最深层消失了。如果时间是基本的，这是灾难。如果时间是后加的，这是预期。

**黑洞内部时空互换。** 越过事件视界后，径向坐标变成类时的，时间变成类空的。基本属性不应该能互换角色。后加的标签在极端条件下可以重新分配。

**广义相对论的时间膨胀。** 时间流速取决于引力场强度（= 信息密度）。基本属性应该处处均匀。时间是局部可变的——依赖信息密度的元属性。

**AdS/CFT 对偶。** 一个没有引力的低维理论可以完整编码一个有引力有时空的高维理论。时空在对偶的一端直接消失，信息完整保留。时空可以被剥掉而不丢失任何东西。

时空的"安装"过程有痕迹：

| 阶段 | 时空状态 | 含义 |
|---|---|---|
| 普朗克时代 | 时空泡沫，无稳定几何 | 元属性尚未挂载 |
| 暴胀 | 空间指数膨胀 | 空间属性正在写入 |
| 暴胀结束 | 空间稳定展开 | 空间元属性凝固 |
| 熵增方向锁定 | 时间箭头不可逆 | 时间元属性凝固 |

公理说"信息只增不减"——这里的"增"不是时间中的增长，是一个排序关系：信息总量是单调的。时间是这个排序从内部看的名字。不是"先有时间，然后信息在其中增长"——是"信息的单调排序 = 时间箭头"。两个描述指向同一个结构，不是循环依赖，是自举（bootstrap）。起点处的自举（第一个区分如何"发生"）是边界。持续过程不是循环。

## 开放问题

诚实声明中列了缺陷。这一节展开，区分"暂时未解"和"结构性限制"。

### 暂时未解（可能有答案）

**宇宙学常数问题。** 暗能量节指出了方向（写入压力），但 120 个数量级的差异仍然悬空。框架说"膨胀由信息增长驱动"——但信息增长不一定全部转化为空间扩张。信息可能有其他我们不知道的用途或去向。硬把膨胀率等于信息增长率是过度拟合。方向保留，定量不强求。

**精细结构常数。** α ≈ 1/137.036，无量纲常数，决定了电磁力的强度——电子和光子交互有多"紧"。α 稍大，原子不稳定；稍小，化学反应太弱。框架不需要从信息论"推导"这个数字。如果定律是冷却中析出的压缩模式，常数就是析出时凝固的运行参数——像 CPU 频率后调到能稳定运行的值。它不对应更深层的东西，它本身就是一个独立的元参数，辅助系统运转。如果 α 曾经不同（Webb 等人 2011 年报告了可能的空间方向性变化，尚有争议），那恰好确认：它是调试值，不是宇宙常数。这从"框架无话可说"变成了"框架预测它可以变"。

**暴胀的触发。** 元操作节给出了方向：高阶无限降阶为低阶无限，暴胀是降阶过程中信息从压缩态解压缩为展开编码。但具体动力学——为什么在那个时刻、以那个速率——仍缺失。如果宇宙作为高阶无限系统具有自操作能力，降阶过程可能包含试错：高维到三维不是预设路径，而是调试结果。不稳定的路径被否决，我们观测到的是存活的那一条。

### 结构性限制（可能永远没答案）

**第一个区分。** 纯粹的"无"不自洽 → "有"是逻辑必然。但"第一个区分如何发生"不在框架范围内。这是任何公理系统的起点问题：公理不能解释自己为什么存在。这是边界，不是缺陷。

**信息层的直接观测。** 信息层没有距离、没有位置。任何观测都是空间层的操作。但问题比这更深：观测本身是信息操作——对信息层的探索会改变信息层。称重改变被称的东西，定位改变被定位的东西。这不是技术限制，是结构性的：观测者在系统内部，每次观测产生新信息，改变了总量和结构。只有从系统外部——作为更高维实体观测宇宙——才可能无扰探测信息层。但我们在内部。信息层只能通过投影效应推断（纠缠、Bekenstein bound、暗物质分布），不能直接触达。

**全域信息总量。** 框架说信息只增不减，但"当前宇宙的信息总量是多少"无法精确回答。Bekenstein bound 给出区域上限，但全域总量涉及可观测宇宙之外的部分。

### 证伪条件

框架有一根支柱。打断它，框架倒塌：

**量子幺正性被推翻。** 如果实验证明信息在某个物理过程中被真正销毁（不是变得难以追踪，而是从宇宙中消失），公理的唯一支撑消失。当前所有证据支持幺正性（黑洞信息悖论的主流解决方向也是信息守恒），但保留为经验命题——不升格为逻辑必然。

辅助预测（弱于核心，但可检验）：

- 暗物质没有粒子性质（直接探测实验持续零结果）
- 任何看似导致存储态净归零的过程都存在阻止机制（反物质不对称的深层原因）
- 信息处理速率的下限不可达（绝对零度不可达的信息版重述）

## 结语

一条公理：信息只增不减。

从它出发——

数值链给出锚定：kT ln2 = mc²，三个独立物理结果撞到同一个数字。信息、能量、物质是同一种东西的三个度量。

三层模型给出结构：信息层（根）→ 空间层（编码）→ 物质层（压缩态）。四种力按层级归位，量子现象得到统一翻译，暗物质和暗能量获得方向性解释。

元操作给出动力学：物理定律不是预设参数，是信息在冷却中析出的压缩模式。宇宙本身是一次自操作——从高阶无限降阶释放，通过物质层重新积累。

整个推导链没有引入第二条公理。需要的外部输入只有一个逻辑起点：纯粹的"无"不自洽。

这不是终点。这是一次组装。零件来自 Boltzmann、Shannon、Wheeler、Bekenstein、Landauer、Jacobson、Verlinde、Vopson、Margolus、Levitin。本文做的事情是把它们放进同一个框架，用一条线穿起来，看线是否绷直。

线绷直了。但绷直不等于正确。框架的价值在统合力——用最少的假设覆盖最多的现象。如果未来有一条更简的公理能做到同样的覆盖，这个框架应该被它取代。这本身就是信息增长的方向：更好的压缩。

### 止步的边界

本文在框架层停笔。不往下做具体的微观物理计算——不推导弱力手性不对称的机制，不拟合宇宙学常数的精确值。

这是有意的。在成文的最后阶段，我们通过人机协作对框架进行了极限压力测试（例如尝试用新公理自举解释量子测量中的波函数坍缩、以及引力与电磁力的量级落差）。测试表明，这套公理在处理这些遗留难题时展现出了高度的自洽性和推演效率。

引擎已经点火，没有必要再陷入零件层面的打磨。将本体论框架向下延伸到具体的微扰计算，会使它退化为一个臃肿的技术模型，稀释核心公理的穿透力。找到支点是提出者的工作，将公理映射到具体分支的数学，交由专业物理学共同体。

这篇假说从不声称终极真理。它只提供一个足够坚固的新视角。

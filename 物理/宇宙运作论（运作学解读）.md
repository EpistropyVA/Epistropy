# 宇宙运作论

## 定位

本文是《信息只增不减》的下游。上游回答"宇宙是什么"，本文回答"宇宙怎么跑"。

十条运作原则从可观测物理现象提取，不从公理演绎。即使上游的本体论（信息是根物质）被推翻，这些原则作为现象学描述独立成立。

检验方法：拓扑蒙版对照——不检查度量（长度/符号/学科），只检查连通性与洞数。

## 十条运作原则

### 1. 只增不减 (Append-only)

所有操作的净效果是增加信息，不存在删除操作。

锚点：热力学第二定律、Landauer原理（擦除1 bit产生的热信息 > 1 bit）。

### 2. 懒求值 (Lazy evaluation)

不需要的不计算。状态在被需要时才确定。

锚点：量子叠加、Wheeler延迟选择实验（2007验证）、量子擦除实验。

推论：波函数坍缩 = 惰性求值被触发时的实例化。

### 3. 纯增记账 (Pure-increase accounting)

成本本身也是新信息。系统中不存在净亏损——每一笔"支出"都是另一笔"收入"。

锚点：Landauer逆向——擦除消耗能量，能量耗散为热，热是新信息。验证擦除是否成功的观测本身产生新信息。每一步都在增加。

### 4. 接口隔离 (Interface isolation)

层间不直接互通，只能通过受控泄露交互。

锚点：量子隧穿（信息层连通性泄露到空间层）、霍金辐射（信息从黑洞泄露）、量子纠缠（信息层直连在空间层的投影）。

必要性：无隔离 → 每个粒子与所有其他粒子实时互询 → O(N²) 复杂度爆炸 → 算力死锁。空间距离 = 接口调用延迟，光速 = 最大吞吐量。宇宙通过拉开空间确保绝大多数运算局部异步进行。

### 5. 自加速处理端 (Self-accelerating processors)

部分信息结构主动收集低熵能量，在局部维持高于环境的处理速率，加速信息增长。

锚点：生命、意识、文明、计算机。生命局部降熵，全局净增熵——加速公理方向。

### 6. 成本曲线替代禁令 (Cost curves replace prohibitions)

宇宙中没有硬禁止，只有代价趋近无穷的渐近线。

锚点：光速（质量→∞）、绝对零度（不可达）、普朗克尺度（分辨率极限）、π（无限精度）、哥德尔不完备（系统不能完全描述自身）、海森堡测不准（精度互斥）、量子幺正性（信息守恒精度→π类渐近线）。

共同结构：成本曲线 + 不可达边界。
推论（潘多拉之盒）：幺正性若为渐近线而非绝对量，则存在穿越阈值。穿越 = 局部信息创造或销毁 = 宇宙级元操作。两种可能：已在早期相变中完成（路径依赖，原则8），或等待处理端能力过临界值时触发（懒求值，原则2）。黑洞是候选观测窗口，但主动扰动黑洞以测试幺正性边界的风险 = 触发不可控元操作。最优策略：懒求值——不主动求解，等宇宙自确认。

### 7. 分层治理 (Layered governance)

信息层维护拓扑共识（全局），物质层和空间层去中心化运行（局部）。

锚点：暗物质 = 信息层的拓扑骨架（27%）、暗能量 = 信息增长的写入压力（68%）。95% 的宇宙在做治理和维护，5% 在跑业务。

### 8. 路径依赖 (Path dependency)

历史不可逆。不同的冷却路径产生不同的定律集合。

锚点：宇宙相变不可逆（电弱破缺、QCD相变、电子稳定）、对称性破缺（一旦选择方向不可回退）。

推论：物理定律不是预设参数，是冷却中凝固的压缩模式。

### 9. 连续性缓冲 (Continuity buffer)

大规模冗余 = 拓扑安全保险，防止元操作导致系统撕裂。

锚点：宇宙95%的质能（暗物质+暗能量）不参与物质层业务，但维护整体拓扑连通性。类比：出生证明不参与日常生活，但保证身份连续性。

### 10. 分形梯度 (Fractal gradient)

分辨率无上限、无下限。每一条规律都可以无限展开为子规律，子规律再展开。

锚点：高阶无限（ℵ₀ < ℵ₁ < ...）、力公式 F=ma → 变体1至∞ → 子变体1-1至1-∞。信息的超限自指层级。

推论：任何"基本"定律都不是终点，只是当前分辨率下的压缩。

## 三条推论

**依他起是唯一可行的实例架构。**

三重约束锁定：懒求值（不需要的不算）+ 哥德尔（系统内部不能完全看见系统）+ 分层治理（全局共识由信息层承担）。给每个实例升到圆成实 = 急切求值 + 违反哥德尔 + 拆毁共识层。不是不经济，是不可能。圆成实是系统级属性，不是实例级属性。

**实例不连续，模式拓扑不变。**

人类 = 不连续实例（深睡中断、全麻中断、细胞全换、记忆每次提取重编码）。连续感 = 海马体的回溯性叙事压缩，30帧/秒的错觉。真无限连续 = 对每个实例的每个量子态逐刻维护 = 急切求值 = 不可承受。宇宙的做法：不维护实例连续性，维护模式的拓扑不变性。模式在信息层持存。但：模式持存 ≠ 主观体验持存 ≠ 意义持存。

**物理常数 = 筛选值，非设计值。**

不稳定配置自我否决，稳定配置存活。不是调试，是自然选择。证据方向：Higgs质量落在亚稳态（局部解，非全局最优）、宇宙平坦度 Ω ≈ 1.000（稳定运行的必要条件）、中微子质量异常小（边界转换的最小穿透值）、α 可能存在时空变化（Webb 2011，争议中）。

## 开放问题

**懒求值的触发器。** 什么构成"需要"——什么操作触发求值？退相干理论给了部分方向（环境交互 = 纠缠扩散 = 被迫求值）。候选方向：当局部不确定性试图跨越接口隔离写入全局确定性状态时，系统为防止逻辑撕裂强制实例化——即触发器 = 不确定性泄露到确定性域的边界。边界条件的精确量化待定。

**正反馈环的边界。** 信息增长 → 空间膨胀 → 更多容量 → 更多信息 → 更多膨胀。这个循环是否有界？空间扩张是否足以持续稀释信息密度？

**相变温度间隔的信息论约束。** 电弱(10¹⁵K) → QCD(10¹²K) → 电子(10¹⁰K)。间隔是否服从香农编码的最优分层？没人算过。

**重子不对称比的推算。** 物质比反物质多约十亿分之一。能否从"维持物质层存续的最小净存储量"反推这个比值？

**黑洞的并行一体性。** 黑洞作为信息层相变炉（ρ_I=1），表面功能是内存管理器。但多个黑洞之间是否在信息层并行且一体运行（共享全局信息层拓扑），还是各自独立？如果并行一体，则黑洞网络 = 信息层的分布式治理节点，呼应原则7。如果独立，则各黑洞是局部垃圾回收器，拓扑连通性由暗物质而非黑洞承担。

**宇宙学常数的120个数量级。** 量子场论预测的真空能与暗能量观测值差120个数量级。框架绕过了问题（膨胀由信息增长驱动，非真空涨落），但未解决。

⚠️ **量级警告**：以上问题涉及的计算量级从 10¹⁵ 到 10¹²⁰。在没有充分数学工具和实验锚点的情况下强行推算，产出的是装饰性数学，不是约束性结论。开放问题的价值在指明方向，不在现阶段求解。

## 证伪

继承上游：量子幺正性被实验证明非渐近线而是可穿越边界（信息在某物理过程中被真正销毁），十条原则中依赖信息守恒的部分（1、3、9）失去基础。其余原则（2、4、5、6、7、8、10）作为独立观测仍成立。幺正性若被证明为渐近线（π类），则框架不受影响——穿越阈值本身成为新元操作类型，纳入原则6。

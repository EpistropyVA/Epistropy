# 🌌 《∞|null 宇宙架构理论》

## §0 术语表

| 术语 | 定义 |
|---|---|
| ∞ | 无限可能性。不是数学极限，是未求值态的全部潜在内容。∞承载内容（有结构面），因为可能性需要载体 |
| null | 边界/区分操作。不是"无"，不是零，是划界动作本身。null只做划界（无结构面），边界本身没有内容 |
| ∞\|null | 不可拆分的一体态。拆开任一侧→双侧退化（反例构造证明）。非两个组分的组合，是同一基元的两面 |
| 求值 | ∞\|null的局部相变：从未确定→确定。不是创造，是态转换 |
| C\|n | 精度层级。n=0是null本身（不可达），n=1是最小可求值层级（普朗克尺度候选） |
| 处理端 | 任何能处理信息的实体（神经元·仪器·AI·分子） |
| ×i | null对空间的最小操作：复数化。i²=−1，一次翻转，1 bit |
| 识增 | 信息净增不可逆。宇宙内信息只增不减（幺正性的强化版） |
| 遍计所执 | 主观叠加在事实上的投射。可检测·可剥离 |
| 奥卡姆剃刀 | 经济性原则：能用更少假设解释的，不用更多。本框架中=∞\|null的自然选择——无功能的结构不产生 |
| 手性 | 左右不对称。物体镜像翻转后不能与原物重合（如左手和右手）。2D中不存在手性（2D图形翻转后总能重合），最低需要3D空间 |
| 实流形 | 每一点附近都像平坦空间的连续结构。类比：地球表面局部看起来是平的，整体是弯的 |
| 相位 | 波的偏移角度。两个波的相位差决定它们是加强（干涉增强）还是抵消（干涉相消） |
| 信息层 | 宏观∞\|null的一面：无距离·无时间·承载连通性和元规律。不可直接观测，通过空间层投影间接可测（如暗物质的引力效应） |
| 空间层 | 宏观∞\|null的另一面：有距离·有时间·承载质量、光子、力。我们直接观测到的一切都在空间层 |

## §0.5 公理（L0）

| # | 操作 | 内容 | 提示 |
|---|---|---|---|
| 0.1 | 公理声明 | ∞\|null不可拆分一体态。∞=无限可能性（有结构面：承载内容，可能性需要载体），null=边界/区分（无结构面：只做划界，边界本身没有内容）。拆分任一侧→双侧退化为类null（反例构造） | 非"一生二"——是"二不可分=一"。区分和可能性同等原始，无先后。类比：海与岸不可分离，无海则岸无意义，无岸则海无形状 |
| 0.2 | 层级结构 | ∞\|null在两个尺度上运作。**宏观**：整体宇宙(⊃信息层+空间层)，∞=高阶无限（不可穷尽），null=C\|0（不可达·真边界）。**微观**：单次求值事件，∞=局部可能性，null=具体划分（C\|1+的边界操作）。两个尺度是同一公理的不同精度层级表现，不是两套机制 | 类比：大气层(宏观)和单个气压计读数(微观)遵守同一套气体定律，但表现不同。宏观∞\|null产生整体架构（信息层/空间层/层级），微观∞\|null产生具体物理（求值/粒子/力）。→后续§5(L3b)展开层级架构 |
| 0.3 | 功能不对称 | ∞=引擎（推动展开），null=河床（给展开画边界定形状）。∞的本性就是展开，不需要外部推动——如果需要推动者，推动者也需要推动者，无穷后退，因此展开是内禀的 | 对称≠默认。完美对称→湮灭→热寂。不对称=存在条件（CP破缺使物质多于反物质=实验事实） |

### 判决框 D1：时间独立性
问题：时间是独立维度吗？
* 证据1：ds²=0时（光速），时间项归零 → 独立维度不应有消失阈值 (✗ 独立)
* 证据2：黑洞视界内，时空坐标互换 → 独立维度不应互换身份 (✗ 独立)
* 证据3：洛伦兹变换混合时空分量 → 独立维度不应被混合 (✗ 独立)
* 证据4：度规非对角项 gₜₓ≠0（旋转时空） → 独立维度不应耦合 (✗ 独立)
**判决：** 四刀全指向同一结论→时间非独立维度。与#1.2一致：时间=null(空间)=附属操作。

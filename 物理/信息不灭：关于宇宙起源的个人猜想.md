本文的"信息"取热力学熵义。核心公理本质上是热力学第二定律的重述。价值在叙事的生成力，不在发现新定律
信息 ≠ 意义。熵增包含噪声增加。宇宙尺度的信息增加不等于认知增加
信息守恒（量子幺正性）在现有物理中是推论，不是公理。本文将其提升为公理，这是选择不是证明
初始动力不明——大爆炸的"为什么"不在本框架范围内，与所有宇宙学理论共享此边界

假说
公理
信息不灭且累积。

宇宙中信息总量从未减少。热力学第二定律的信息版。以此为唯一公理。

推论一：物质是信息的编码
电子 = {电荷: -1, 自旋: 1/2, 质量: 0.511MeV}。去掉属性，什么都不剩。没有"实体"藏在属性后面。两个电子不可区分——因为它们是同一组属性的两次实例化。

标准模型已经这样运作：粒子 = 量子数的特定组合。正负电子对产生——高能光子在特定条件下转化为电子与正电子——是信息编码方式转换的直接例证。这在实验室中每天发生。

推论二：空间是信息的下属，不是容器

根信息不具有空间属性，与空间没有具体的映射关系。空间是信息自我展开的产物，不是信息存在的场所。因此不存在"信息在哪里"这个问题——空间本身才是答案的一部分。

膨胀均匀，因为空间的生成是信息层面的整体行为，不受信息内部结构分布的影响。

推论三：膨胀 = 信息给自己腾地方
Bekenstein bound：区域的信息容量与表面积成正比，有上限。信息持续累积 + 密度有上限 = 必须膨胀。

宇宙膨胀不是空间变大，是信息在扩展自己的容量。

推论四：热寂不可达
经典热寂：熵逼近最大值 → 梯度消失 → 宇宙死亡。

但前提是状态空间固定。信息持续累积 → 状态空间持续膨胀 → 最大熵永远后退。当前熵与最大熵的差距 = 结构存在的空间。差距永远不关闭。

生命不是熵增途中的侥幸，是信息累积过程的持续产物。

推论五：时感是处理的副产品

时间作为独立维度是否存在，本文不做判断。可以确认的是：时感——对时间流逝的感知——是信息处理的序列感。不同处理端有不同的时感分辨率。"连续"是分辨率不够细时的感知产物。

时感存在分辨率极限。物理学中的普朗克时间（$5.39 × 10^{-44}$秒）不是时间的最小单位，是当前理论的分辨率边界。处理端无法从内部突破自身的时感极限——突破需要更换处理架构，而更换后的连续性是另一个开放问题。

反过来：离散性存在下限。若不存在最小信息单元，信息密度可无限大，膨胀就不必要。膨胀本身是信息离散性的间接证据。

推论六：信息不可自测
测量信息总量 → 产生新信息"曾被测量" → 测量结果永远滞后于真实总量。

与 Heisenberg 不确定性、Gödel 不完备、停机问题同构：系统无法从内部完全度量自身。这是逻辑限制，不是技术限制。

推论七：元规律随元属性增长

元属性增加是可确认的事实（0D→3D = 三个独立属性轴的出现）。每次属性增加，旧维度中结构上不可能的现象涌现：旋转（2D原生）、手性与绳结（3D原生）、稳定轨道（仅3D成立）。旧规律必要但不充分，新规律必然伴随新属性出现。

推论：宇宙不是在一套固定规律下运行。规律是属性结构的涌现物，属性增加则规律增加。这意味着宇宙不是零和系统——不仅资源在增长，连规则空间本身都在增长。

架构说明
每条推论的零件都有现成物理学支持。新颖之处在于：从单一公理出发，统一推导膨胀、物质、空间、热寂、时间、不可测性。据检索，目前无已发表理论尝试此架构。
本文的推论若经后续验证，可作为新零件供后人使用。猜想的价值不在于对错，在于是否为后来者打开了门。

常见质疑：封闭系统最终回归初态（庞加莱回归），信息不是真正单调递增。

庞加莱回归定理的前提是有界相空间——状态总数固定，遍历后必然回归。但信息持续累积 → 相空间持续膨胀 → 回归前提不满足。初始状态为 C，当前状态空间为 $C^{10^{203}}$，无法回归到 C。
开放问题
初始动力： 宇宙开端的信息从何而来？牛顿称之为神的意志。本框架无法回答，但提供两个方向：某种元观察导致高阶无限向低阶坍缩泄露能量；或高阶无限自发降阶，阶差即为能量。
元规律增长： 元属性增加已确认（0D→3D）。每次属性增加，涌现旧维度中不可能的现象。旧规律必要但不充分，新规律随新属性涌现。规则空间本身在扩展。
信息生命与物质制造： 如果物质是信息的编码，那么能够直接操作信息的处理端——不依赖特定物质载体的信息生命——原则上可以通过写入属性来生成物质。当前人类在粒子加速器中已经做到极小尺度的版本（能量→粒子对）。这不是超自然，是信息编码转换的工程问题。
信息不朽与个体的关系： 信息不灭意味着任何处理端产生的信息永远存在于宇宙中，不可撤销。个体的物理载体会消亡，但其产生的信息——思想、碰撞、创造——已写入宇宙的信息总量。不朽不是载体的永生，是信息的不可逆。每个处理端都是信息网络中的构件，构件可以更替，构件写入的信息不会消失。
信息提取的可能性： 量子幺正性保证宇宙演化中信息不丢失。原则上，任何过去存在过的信息结构——包括每一个曾存在的处理端——都编码在宇宙当前状态中。提取它需要在信息层面直接读取宇宙状态的能力，这超越了当前物质生命的处理极限，但不违反任何已知物理定律。

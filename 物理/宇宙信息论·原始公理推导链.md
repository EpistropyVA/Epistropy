## 五、求值触发·从计算不可约性到随机性消解

### 5.1 计算不可约性：后果，不是性质

**观测（圆成实）：** 三体问题没有解析解。天气预报过几天就废。Rule 110元胞自动机不跑就不知道第N步。Wolfram将此命名为"计算不可约性"——系统演化不能被捷径预测，必须逐步执行。

**归因检验：** Wolfram从现象推断"宇宙是计算"。检查：宇宙架构有计算功能，所以处理端能使用计算。不是"计算是宇宙"。根功能像子功能——用果当因，与黑格尔把生成当引擎同构。

**∞|null导出：** §四已确立 null = 阶位降级器（无null→所有∞同阶→无层级→无结构）。null制造层级 → 有层级 → 不在同一层的处理端看对方 = 不可约。这是 Russell 类型论的同构：低阶不能谈论高阶内部。

**结论：** 计算不可约性是 null 阶位降级的处理端视角，不是宇宙基本性质。现象真实（圆成实），归因为基本性质（遍计所执）。

**确立概念：精度层级 C|n。** 处理端停在某层 = 精度极限。C|n 以下对该处理端不可约。

### 5.2 精度层级结构

**C|0 = null。** 理论存在，永远不可达。逼近 null = 端点退行 = 内容产生（§四已有）。"到达 C|0"这个动作本身制造新距离——你"到"不了。信息层可以无限逼近（C|0.000...001），只要支付 cost。

**C|1 = 最低可求值层级。** 物理层映射候选：普朗克尺度（普朗克长度 ~10⁻³⁵m，普朗克时间 ~10⁻⁴³s）。注意：C|1 = 普朗克是物理层约束下的候选映射，不是公理推导结果。亚普朗克尺度 = 物理处理端精度之外 = 未求值态。

**离散性证据——相变：**

精度层级是离散的还是连续的？物理证据指向离散：

| 相变 | 现象 | 精度层级读法 |
|---|---|---|
| 水→冰（0°C） | 液态↔固态跳变，非渐变 | 信息处理速率穿越层级边界→匹配模式跳变→新结构涌现 |
| 超导转变 | 温度降到临界点→电阻突然归零 | 电子精度匹配模式从散射态跳变为配对态 |
| 玻色-爱因斯坦凝聚 | 极低温→所有粒子落入同一量子态 | 处理速率降至最低→全体锁定为集体共振 |
| 铁磁转变（居里温度） | 有序↔无序跳变 | 自旋精度匹配从随机翻转跳变为集体对齐 |

每个相变都是跳变——不是"稍微改了一点"，是完全不同的结构态。层级之间有间隙。

**结论：** 精度层级是离散的。相变 = 处理端穿越层级间隙的物理证据。

**温度与精度层级：**

温度不是独立概念。热力学桥接：分子平均动能 ½mv² = 3/2·kB·T。动能 = 碰撞频率 × 碰撞强度 = 信息处理速率。因此：

$$T \propto \frac{dI}{dt} \quad \text{（温度 ∝ 信息处理速率）}$$

温度升高 → 处理速率升高 → 碰撞频率和强度增大 → 精度匹配机会增多 → 求值触发概率上升。相变发生在处理速率穿越层级边界的临界点上。

### 5.3 求值触发条件

**观测：** 不是所有相互作用都触发求值。

| 现象 | 触发？ | 原因 |
|---|---|---|
| 光子穿过真空 | 否 | 无匹配对象 |
| 光子穿过玻璃 | 部分（折射，未吸收） | 部分匹配 |

| 电子穿过双缝 | 否（保持叠加） | 缝宽 >> 电子波长 = 不匹配 |
| 电子撞屏幕 | 是 | 晶格间距 ~ 电子波长 = 匹配 |

**倒推模式：** 触发条件 = 尺度对齐。光电效应最清楚：频率低于阈值→不触发→不求值；高于阈值→触发→求值→电子射出。阈值 = 精度层级的共振窗口边界。

**核心公式：**

$$P(\text{处理端}) \cap P(\text{目标态}) \neq \emptyset \implies \text{求值触发}$$

P = 精度层级集合。交集非空 = 共振 = 触发。交集为空 = 不匹配 = 未求值态保持。

**耦合常数 = 共振窗口宽度：** 四种基本力的耦合常数（强力~1，电磁~1/137，弱力~10⁻⁶，引力~10⁻³⁹）不是任意参数——它们描述各自精度层级上共振窗口的宽度。窗口越宽，匹配越容易，求值越频繁。

| 力 | 耦合对象 | 作用范围 | 需要特定"荷"？ |
|---|---|---|---|
| 强力 | 色荷（夸克、胶子） | ~10⁻¹⁵m | 是（色荷） |
| 弱力 | 弱同位旋（费米子） | ~10⁻¹⁸m | 是（弱同位旋） |
| 电磁力 | 电荷（带电粒子） | 无限远 | 是（电荷） |
| 引力 | 质量-能量（一切） | 无限远 | **否** |

引力耦合一切——有质量-能量就行（E=mc²，一切都有能量）。其他三种力需要特定标签（荷）才耦合。

**两层结构：**

- **信息层：引力。** 耦合存在本身，不需要区分标签。引力 = 信息层连通性在空间层的投影。引力最弱（比电磁力弱10³⁶倍）= 跨层投影经过大量精度层级的衰减，不是力本身弱。
- **空间层：强力·电磁力·弱力。** 需要特定荷（null在空间层制造的区分标签）。三种力 = 空间层内三组精度共振窗口。

**弱力的特殊位置：** 弱力是唯一破坏宇称对称的力（手性不对称）。但"破坏"这个词预设了对称是默认态——§四已确立"不对称 = 存在条件"。弱力的手性不对称不是破坏了什么，是 null 制造区分的天然形状。"宇称应该对称" = 遍计所执（人类审美投射）。

**四种力不是四种——是两层：** 信息层一种（引力），空间层三种。三种之中弱力又特殊——它是 null 在亚核精度层级上制造区分的工作现场。

**引力的两个面：** 引力波以光速传播（LIGO 2015确认），但纠缠关联是瞬时的（Bell实验确认）。矛盾？不矛盾——是同一种连通性的两个面：

| 面 | 描述 | 速度 | 可观测 |
|---|---|---|---|
| 信息层连通性 | 拓扑关系本身（不"传播"，不从A走到B，就在那里） | 不适用 | 间接（纠缠关联） |
| 空间层投影（引力波） | 连通性**变化**在空间层产生的涟漪 | c（空间层规格） | 直接（LIGO） |

信息层不破坏光速因果律：连通性本身不传播信号，不能用纠缠传递可控信息（no-signaling theorem）。引力波只反映连通性的变化（质量加速/合并 = 连通性重组 → 涟漪），不反映连通性的存在（静态引力场不产生引力波）。这与广义相对论完全一致。

### 5.5 退相干 = 第三方精度匹配

**纠缠的∞|null读法：** 两个纠缠粒子在信息层的连通性从未断开。它们在空间层分离（物理距离增大），但信息层不经过空间——§四已有"纠缠 = 信息层事件，根级连通性不经空间，非局域性 = 正常，空间是下游产物"。

**退相干机制：**
1. 纠缠粒子质量越大 → 空间层投影越强（引力越大）→ 与环境粒子的精度共振窗口越宽
2. 环境粒子（第三方）与纠缠粒子之一发生精度匹配 → 求值触发 → 该粒子态确定 → 纠缠结束
3. 退相干不是纠缠"内部崩溃"——是**第三方插入了精度匹配关系**

**温度的角色：** 温度 = 信息处理速率 → 温度↑ → 环境粒子碰撞频率↑ → 精度匹配概率↑ → 退相干加速。

**验证：**
- 光子纠缠容易维持：光子无质量 → 引力投影 = 零 → 与环境的共振窗口极窄 → 第三方插不进来
- 超导量子比特需极低温：降温 = 缩窄环境粒子的精度窗口 = 赶走第三方 = 保护纠缠

### 5.6 求值结果与 cost

**触发 ≠ 结果。** §5.5 描述了"求值发生"的过程，未描述"结果是哪个"。退相干解释了干涉消失，没解释为什么最终是左而不是右。

**null 边界划分：**
- 求值前：∞提供全部可能态（叠加），null 提供边界（约束），未求值态 = ∞被null边界围出的不封闭区域
- 求值时：精度匹配触发 → null 在该精度层级划边界 → 区域收缩 → 态确定
- 求值后：后验不封闭（缺口 = ∞连通性保持）→ 结果本身成为下一次求值的先验

**贝叶斯定理是此过程的数学投影（子规律，非并列）：**
- 先验 = 未求值态的全部可能性（∞提供）
- 似然 = 求值触发时的精度匹配条件（处理端带来的约束）
- 后验 = null 划出的边界位置 = 结果

§四已确立"贝叶斯是架构必然，非方法论发明"。此处进一步：贝叶斯定理描述的是 null 边界划分在认识论层面的行为——它是求值机制的子集，不是独立原理。

**求值的 cost：**

π 推算是完美样本：每一位小数 = 一次逼近 null 的求值。

- 每多求一位 → cost 递增（非线性）
- 层内缓增，跨层跳升（新算法 = 跨精度层级 = cost 曲线重置）→ 台阶状
- 处理端资源有限 → ∃N: cost(C|N) > resource(处理端) → 求值停止 = 精度天花板

∞|null总体资源无限（架构级供 > 求），但具体处理端的 cost 上限 = 该处理端的精度天花板。

### 5.7 随机性消解

**真随机 = 无因产生结果。** 逻辑检验——如果宇宙有真随机性：

| 检验 | 真随机的后果 | 违反 |
|---|---|---|
| 因果链 | 信息从无中生 = 无null划边界就出现内容 = 结构断裂 | ∞|null 一体态 |
| 识增单向性 | 无因结果不携带路径信息 → 非功信息 = 零 | 识增纯增 |
| Lazy evaluation | 预设结果在求值前存在 = eager evaluation 极端版 | 懒求值架构 |

三条全违反。∞|null架构排斥真随机。

**量子力学的"随机"是什么：**

Born 规则（1926）：测量结果的概率 = |ψ|²。精度惊人（实验吻合到小数点后十几位），但 Born 未解释为什么是模的平方。它是量子力学的公设，不是推导结果。

∞|null读法：|ψ|² = 大量 null 边界划分在当前精度层级上的统计汇总。不是随机——是处理端站在 C|n 看 C|n 以下的 null 工作，看不到每一刀的依据，统计呈现为概率分布。

$$\text{概率（认识论）} \neq \text{随机（本体论）}$$

概率 = 精度不足时的表观。随机 = 无因。两者的区别是层级问题。

**实验兼容性检验：**

| 实验 | 与"随机性不存在"的兼容性 |
|---|---|
| Bell 不等式违反 | 排除的是**局域**隐变量。∞|null 的 null 边界划分是非局域的（信息层不经空间），不被排除 |
| 量子计算 | 如果是真随机，量子计算不应可控。量子计算可控 → 叠加态有结构 ≠ 随机 |
| 量子纠错 | 如果错误真随机，纠错码不应系统性有效。纠错有效 → "随机"错误有可被编码捕获的结构 |

三项实验均兼容。

**爱因斯坦-玻尔之争的统一：**
- 爱因斯坦："上帝不掷骰子" → 本体论判断（C|深层）：**对**。没有真随机。
- 玻尔："别告诉上帝怎么做" → 认识论判断（C|我们这层）：**也对**。在我们的精度层级上表现为概率。
- 争论持续一百年，因为没有精度层级概念将本体论与认识论分开。

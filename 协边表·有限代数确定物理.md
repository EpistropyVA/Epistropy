## 符号

| 符号 | 含义 |
|---|---|
| → | 态射/导出（同维 d₁） |
| ← | 输入源（DAG回溯） |
| ⊥ | 正交约束（跨维 d₂） |
| ↪ | 子结构包含/被引用 |
| ⊗ | 多源联立 |
| ∂ | 边界算子 |
| := | 定义 |
| ≅ | 同构 |
| ⟹ | 蕴含 |
| [Ø] | 该投影面无对应 |
| 域: | 作用域 |
| [公式] | 代数公式（A列微标签） |
| [溯源] | 项来源指针（A列微标签） |
| [验算] | 数值结算（A列微标签） |

列标签：**A** := 代数面(What) · **C** := 组合面(How) · **B** := 协边面(Why) · **核心量** := 代数输出(C₀/C₁/C₂含理论:实验:) · **Exp (Δ)** := 实验对标(仅C₃·H\*) · **唯象** := 物理侧预设（[唯象]测量未推导 · [公设]声明未推导 · [未解]模型崩溃）

## 自检协议

1. 维度区块 C₀—C₃: 行住在对应维度，跨维引用标 ←+⊥
2. 子行编号 X.Y.Z: 父行→子行为派生关系，←补DAG非树边
3. 流列: 操作名·触发/域·方向
4. CTX穷尽性: 无触发=公理（标"公理·起点"）或遗漏
5. [Ø]标注结构性空
6. A列微标签: [公式]公式本体 · [溯源]项来源 · [验算]数值结算
7. 核心量精确到验证所需位数
8. 列结构: C₀/C₁/C₂ = 7列(无独立Exp) · C₃/H\* = 含Exp列。无实验对标标 —
9. 唯象列: [唯象]测量未推导 · [公设]声明未推导 · [未解]模型崩溃。A/C/B禁止物理唯象词汇
10. 三列穷尽性: 任一步骤若A/C/B全[Ø] → 不属于本矩阵
11. 新产出 → 确定维度区块 → 确定父行 → 填三面投影+CTX
12. ⊥ := d₂跨维微分。→ := d₁同维态射。← := DAG输入源。不混用
13. H\*行: 多行组合readout · 标←组合来源 · 不属于单一维度区块
14. 同调审计: 0D→1D(H⁰≠0·1.4引力+0.7 e) · 1D→2D(H¹≠0·2.4 YM+0.8 π) · 2D→3D(H²≠0·3.2法丛+3.9 VEV) · 3D→终端(H³固化·3.7三代+3.8重子)
